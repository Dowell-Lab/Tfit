Detailed Description

ParamWrapper parses and preprocesses command line arguments.

The ParamWrapper class was implemented to enable a greater degree of abstraction within tfit with respect to how command line arguments are passed and parsed. Under the previous model, each function that made use of values gathered from the command line would search through a dictionary of argument-value pairs. This meant that any changes to the command line specification of Tfit would require functions throughout the codebase to be revised.
