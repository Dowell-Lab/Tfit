Computes a baysean information criterion score (in two dimensions?) given the input data. As the model optimizes parameters, this function provides a score with which the model can determine the magnitude of improvement.

In effect, the BIC3 function should determine how well a given iteration of the model fits its input data.

The BIC is defined as ln(n)*k-2ln(L), where L is the maximized likelihood function.

This likelihood function is defined as: L=p(x|th, M), where th is the set of parameters that maximize said likelihood function.

In this case, p() is represented by a call to the functions within the EMG class, as EMG encapsulates the distribution fitting code central to tfit's operation.
